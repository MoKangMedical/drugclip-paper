💊 DrugCLIP Contributions

A contrastive learning framework for virtual drug screening with state-of-the-art performance on molecular property prediction benchmarks.

Key Contributions

Research Contributions & Methods

DrugCLIP introduces novel approaches to molecular representation learning and virtual drug screening.

🔗

Contrastive Molecular Learning

Novel contrastive learning objective that jointly learns molecular representations from SMILES strings and 2D/3D molecular graphs, aligning chemical structure with biological activity.

  • CLIP-inspired molecular pre-training
  • Multi-modal molecular encoder
  • InfoNCE loss with molecular augmentations
🎯

Cross-Modal Alignment

Aligns molecular embeddings with protein target embeddings in a shared latent space, enabling zero-shot virtual screening against novel targets without task-specific fine-tuning.

  • Protein-language model integration
  • Shared embedding space (512-dim)
  • Cosine similarity scoring
📊

Comprehensive Benchmark Suite

DrugCLIP Benchmark — a standardized evaluation suite spanning 15 molecular property prediction datasets covering ADMET, toxicity, binding affinity, and activity prediction tasks.

  • 15 standardized benchmark datasets
  • Consistent evaluation protocol
  • Leaderboard with reproducible results
🧪

Molecular Augmentation Strategy

Domain-specific augmentation techniques for molecular contrastive learning including SMILES enumeration, subgraph masking, bond deletion, and 3D conformer perturbation.

  • SMILES canonical augmentation
  • Graph-level structural perturbation
  • 3D spatial augmentation
🔍

Zero-Shot Virtual Screening

Screen millions of compounds against novel protein targets without any activity data for that target — leveraging the learned cross-modal alignment.

  • No target-specific training data needed
  • Screen 10M+ compounds in minutes
  • Hit rate improvement of 3-5x over random
🧠

Interpretable Predictions

Attention-based interpretability that highlights which molecular substructures and protein residues drive the predicted binding, providing mechanistic insights.

  • Atom-level attention visualization
  • Binding site residue highlighting
  • Substructure contribution scoring
📈

SOTA Benchmark Performance

Achieves state-of-the-art or competitive results on 12 of 15 benchmarks, with significant improvements on binding affinity prediction (MUV, DUD-E) and ADMET tasks.

  • SOTA on MUV, DUD-E, Tox21
  • Competitive on all ADMET endpoints
  • Consistent cross-benchmark improvement
⚙️

Scalable Architecture

Efficient transformer-based architecture with linear attention that scales to large molecular libraries while maintaining inference speed suitable for high-throughput screening.

  • Linear attention complexity O(n)
  • Batch inference on GPU clusters
  • ONNX export for deployment
🔄

Transfer Learning Framework

Pre-trained DrugCLIP representations transfer effectively to downstream tasks with minimal fine-tuning, reducing labeled data requirements by 10x.

  • Few-shot learning capability
  • Frozen encoder + lightweight head
  • 10x less labeled data needed
🌐

Open-Source Ecosystem

Complete codebase, pre-trained models, benchmark data, and evaluation scripts released for reproducibility and community adoption.

  • PyTorch + HuggingFace integration
  • Pre-trained checkpoints available
  • Jupyter notebook tutorials

Model Architecture

Technical details of the DrugCLIP framework and training methodology.

Model Architecture

Molecular EncoderGraph Transformer (12L)
Protein EncoderESM-2 (8L frozen)
Embedding Dimension512
Attention Heads8
Total Parameters48M

Training

Pre-training Data2M molecules, 500K proteins
Contrastive Pairs10M positive pairs
Batch Size2048
Training Time72h on 8×A100
OptimizerAdamW (lr=1e-4)

Inference

Single Molecule0.8ms
Batch (10K molecules)1.2s
Library Screen (1M)~2 min (A100)
Memory (inference)2GB GPU
Export FormatsPyTorch, ONNX, TorchScript

Benchmarks

Datasets15 standardized
TasksClassification + Regression
Evaluation5-fold CV + scaffold split
MetricsAUROC, AUPRC, RMSE, R²
